# PublicationsDB
Για την χρήση της εφαρμογής αποθήκευσης και ανάκτησης άρθρων και επιστημονικών περιοδικών είναι απαραίτητη η εγκατάσταση των εξής:

•	Λογισμικό διαχείρισης βάσεων δεδομένων (προτείνεται το DB Browser for SQLite)

•	Γλώσσα προγραμματισμού (πχ. Python 3.10) και εγκατάσταση βιβλιοθηκών sqlite3 και hashlib

•	Visual Studio Code για την δοκιμή της εφαρμογής

Μετά την εγκατάσταση των παραπάνω, ανοίγετε την εφαρμογή DB Browser for SQLite και εισάγετε την βάση επιλέγοντας “Open Database”. Στη συνέχεια επιλέγετε το αρχείο της βάσης (PublicationsDB.db). Αφού γίνει αυτό θα μπορείτε να έχετε πρόσβαση στους πίνακες και τα δεδομένα της βάσης επιλέγοντας το “Database Structure”. Με το “Browse Data” μπορείτε να δείτε όλες τις εγγραφές των πινάκων.
Για την εκτέλεση των queries, επιλέξτε την καρτέλα “Execute SQL”, κάντε αντιγραφή-επικόλληση τον κώδικα και πατήστε Ctrl+Enter. Το αποτέλεσμα θα εμφανιστεί στο κάτω μέρος της οθόνης.
Για την δοκιμή της εφαρμογής, είναι σημαντικό να βρίσκονται στον ίδιο φάκελο το αρχείο της βάσης (PublicationsDB.db) και τα 3 αρχεία κώδικα σε Python. Η εφαρμογή ξεκινάει επιλέγοντας “Run Python File” από τον controller και τότε εμφανίζεται στο τερματικό το αρχικό μενού της εφαρμογής. Από εκεί μπορείτε να επιλέξετε κάποια από τις διαθέσιμες ενέργειες και να πλοηγηθείτε στην εφαρμογή.
